## 📌 Sobre o Projeto

Este projeto explora o uso da **IA Generativa como ferramenta de apoio ao aprendizado**.

A proposta foi criar um **caderno temático utilizando o NotebookLM**, reunindo fontes abertas sobre um **tema financeiro introdutório**. A partir dessas fontes, foram desenvolvidos resumos, perguntas estratégicas e testes de prompts para aprofundar o entendimento do conteúdo.

O projeto demonstra como a inteligência artificial pode ser utilizada de forma **crítica e organizada para apoiar estudos**, transformando informações em um material estruturado para revisão.

---

## 🎯 Objetivos do Projeto

- Reunir **3 a 5 fontes abertas** (textos ou PDFs) sobre um tema financeiro.
- Utilizar o **NotebookLM** para analisar e organizar as informações.
- Criar **resumos estruturados** para facilitar o estudo.
- Elaborar **perguntas estratégicas** sobre o conteúdo.
- Testar diferentes **prompts com IA** e registrar os resultados.
- Construir um **miniguia de estudo reutilizável**.

## 🤖 Uso da IA Generativa

A inteligência artificial foi utilizada como **ferramenta de apoio ao estudo**, ajudando em tarefas como:

- síntese de conteúdos  
- explicação de conceitos financeiros  
- geração de perguntas para revisão  
- organização das informações  

Durante o processo, as respostas da IA foram **comparadas com as fontes originais**, promovendo **pensamento crítico e análise das informações**.
